Run Test Application
1.	Download Mail-Test-Connection.zip file from here.
2.	Unzip Mail-Test-Connection.zip
3.	Open Mail-Test-Connection folder and run run.bat.
4.1.	Select POP3 tab.(Table with two columns: key and value will be displayed)
4.2.	Insert your mail pop3 parameters in this table (Double click on values row(s)).
4.3.	Click “SvaePOP3 Properties” button. System must show information dialog “Properties Saved” -> click ok.
4.4.	Click “TEST|pop3 read mails” button.
4.5.	Select IMAP tab. (Table with two columns: key and value will be displayed)
4.6.	Insert your mail imap parameters in this table (Double click on values row(s)).
4.7.	Click “Save IMAP Properties” button. System must show information dialog “Properties Saved” -> click ok.
4.8.	Click “TEST|imap read mails” button.
4.9.	   Select POP3 tab. (Table with two columns: key and value will be displayed)
4.10.	Insert your mail SMTP parameters. (Double click on values row(s))
4.11.	Click “Save SMTP Properties” button. System must show information dialog “Properties Saved” -> click ok.
4.12.	Type Test Persons email address.
4.13.	Click “TEST|Send Mail” button.
